# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. You'll create a Hangman game where players guess letters to reveal a hidden word before running out of attempts, practicing string manipulation, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Game Core

#### Description
Create a Hangman game that randomly selects a word from a predefined list and allows players to guess letters. The game should track guesses, display progress, and determine when the game ends.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses from the player
- Display the current word progress in underscore format (_ _ _)
- Track the number of incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages
